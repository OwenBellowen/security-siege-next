# security-siege-next
A rewrite of my Discord bot - Security Siege

## Installation

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file in the root directory and add the following:
```
TOKEN=YOUR_DIS
```
4. Run `npm start`
